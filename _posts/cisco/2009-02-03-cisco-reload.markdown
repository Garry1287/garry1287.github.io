# Reload cisco роутер
Перезагрузка:

просто перезагрузка

`reload`

перезагрузка в определенное время:

`reload at hh:mm [month day] [reason]`

перезагрузка через определенное время:

`reload in [hh:]mm [reason]`

отмена отложенной перезагрузки:

`reload cancel`
